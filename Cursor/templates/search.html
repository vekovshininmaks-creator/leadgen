Поиск каналов

Поиск публичных каналов и супергрупп. Отметьте строки и добавьте в мониторинг; ненужные можно отправить в чёрный список (колонка «ЧС») — они перестанут попадать в выдачу. Закрытые чаты в глобальном поиске не появляются.

Важно: перед поиском нажмите «Стоп» на странице «Управление» — одна сессия Telegram.

Чаты из аккаунта

Все каналы и группы из диалогов выбранного аккаунта (или сразу нескольких, если отмечены чекбоксы выше) добавляются в target_chats (фоновая синхронизация, без глобального поиска; при нескольких аккаунтах список объединяется без дублей).

Ключевые слова для поиска

По одному запросу на строку. Можно вписать вручную или сгенерировать через LLM («Настройки и лимиты» → «Подключения»).

Запуск поиска

Рекомендуем: 1000 подписчиков и 14 дней без поста.

рек. 1000
рек. 14

Каналы из комментариев и «о себе»

Для каждого канала из выдачи поиска: открыть привязанное обсуждение, взять авторов комментариев, прочитать их профиль и добавить в результаты каналы/чаты по ссылкам в описании (t.me/…, @username). Медленнее и больше запросов к Telegram.

Подробнее о фильтрах и скорости

Порог по дням требует запроса к каждому каналу; «дата без фильтра» — медленнее (~0,12 с на канал), только для просмотра дат.

По каждой непустой строке — отдельный поиск; дубликаты сливаются. С «1000 + 14 дн.» из сырой выдачи обычно остаётся меньше «живых» каналов.

Блок «Каналы из комментариев» работает только у каналов с включённым обсуждением; без привязанного чата комментариев шаг пропускается. Чекбокс «только каналы с обсуждением» фильтрует всю выдачу (ещё до разворота комментариев).
