Лиды по сайтам

Поиск по поисковой выдаче (SerpAPI / Google) или импорт списка доменов. Скрипт сам тянет главную и страницу контактов, достаёт email/телефон/Telegram/WhatsApp и складывает в воронку.

Ключ SerpAPI не задан — поиск по выдаче выключен. Импорт CSV/доменов работает в любом случае. Сохраните ключ ниже и страница перезагрузится.

Подключения

SerpAPI — платный сервис доступа к поисковой выдаче Google (и др.). Регистрация на сайте, в личном кабинете копируется поле API key и вставляется ниже. Без ключа блок «Поиск по выдаче» не вызывает SerpAPI; импорт доменов/CSV работает как обычно.
