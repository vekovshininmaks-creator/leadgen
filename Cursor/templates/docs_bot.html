Инструкция: бот и воронка

Краткий порядок действий после входа в консоль. Технический деплой на VPS: файлы docs/runbook-deploy-by-ip.md и docs/деплой-сервер.md в каталоге проекта.

  1. Настройки и лимиты → вкладка «Подключения»: api_id и api_hash с my.telegram.org, при необходимости LLM. Вкладка «Воронка и лимиты»: ключевые слова, шаблоны этапов 1–3, лимиты и расписание. Для проверки без реальных ЛС включите «тестовый запуск» (dry-run).
  2. Код из Telegram — после нажатия «Старт» на странице бота введите код в левой колонке («Код из Telegram»), он доступен с любой страницы после входа.
  3. Поиск каналов — добавьте чаты в мониторинг или на странице бота используйте «Синхр. чаты из аккаунта».
  4. Управление ботом — старт/стоп, журнал прохода, последние лиды (подсветка триггера, пометка «шлак» хранится в браузере).
  5. Лиды — таблица и выгрузка sent_leads.csv; ссылки на чаты и сообщения в Telegram при наличии данных в журнале.
  6. Офферы — при включённом согласовании черновики этапов попадают в очередь перед отправкой.
  7. Переписка / Созвоны — ответы по лидам и учёт звонков.
  8. Лиды по сайтам — опционально: ключ SerpAPI на serpapi.com для поиска по выдаче.
